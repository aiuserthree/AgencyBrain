#### 기능 설명

1. 헤더 영역
- BI : 메인 페이지 링크 
- 개인화 아이콘 : 로그인 상태인 경우 개인화 아이콘 노출
  ㄴ 개인화 아이콘 노출 페이지 : 메인, 상품리스트, 혜택+홈, 마이 홈 

2. 브랜딩 영역
- 서비스 알아보기 : 클릭 시 서비스 소개 페이지 이동

3. 추천 상품 리스트 노출 영역
- 추천 상품영역 
  ㄴ 추천 상품 이미지 : [상품관리] 등록 이미지
  ㄴ 추천 상품명 : [상품관리] 등록 상품명 
  ㄴ 추천 상품 설명 : [상품관리] 등록 상품 한줄 설명 
  ㄴ 리스트배너영역 클릭 시 해당 상품 상세로 링크
  ㄴ 더 많은 상품 보러가기 링크 : 클릭 시 상품리스트로 이동

4. 공통영역
- 제휴 카드 혜택 영역 : 노출 기간에 해당되고, 노출 여부가 '노출', 메인/상품 상세 노출 여부가 '노출'이며 우선순위가 높은 순으로 노출. 동일 우선순위인 경우 최신 등록일 순으로 노출
  ㄴ 제휴 카드 메인 썸네일 이미지 : [제휴 카드 관리] 등록 모바일 메인/상품 상세 썸네일 이미지 호출
  ㄴ 배너 영역 클릭 시 해당 제휴 카드 상세 페이지로 이동

- 주력 판매 상품 홍보 영역 : 노출 기간에 해당되고, 노출 여부가 '노출', 메인 노출 여부가 '노출'이며, 노출 우선 순위가 가장 높은 1건의 정보를 노출
  ㄴ html map : 등록된 이미지와 html map 호출

- 진행 중인 이벤트 : 노출 기간에 해당되고, 노출 여부가 '노출', 메인 노출여부가 '노출'이며 우선순위가 높은 순으로 노출. 동일 우선순위인 경우 최신 등록일 순으로 노출
  ㄴ 이벤트 메인/상품 썸네일 이미지 : [이벤트 관리] 등록 모바일 메인 섬네일 이미지
  ㄴ 배너 영역 클릭 시 해당 이벤트 상세 페이지로 이동

- 제휴 혜택 안내 영역 : 관리자 공통코드에 등록된 제휴혜택 카테고리 중에서 ‘노출’상태인 공통코드를 우선순위가 높은 순으로 노출. 
  ㄴ 카테고리명 : 관리자 등록 공통코드명 호출 
  ㄴ 카테고리 이미지 : 공통코드 내 등록된 모바일 이미지 호출 
  ㄴ 배너 영역 클릭 시 해당 제휴혜택 카테고리 리스트 페이지로 이동
  ㄴ 더 많은 제휴 혜택 보러가기 링크 : 클릭 시 제휴혜택 리스트로 이동 

- 전환 상품 안내 영역 : 관리자 공통코드에 등록된 전환상품 카테고리 중에서 ‘노출’상태인 공통코드를 우선순위가 높은 순으로 노출. 
 ㄴ 카테고리명 : 관리자 등록 공통코드명 호출 
 ㄴ 카테고리 이미지 : 공통코드 내 등록된 모바일 이미지 호출 
 ㄴ 배너 영역 클릭 시 해당 전환상품 카테고리 리스트 페이지로 이동
 ㄴ 더 많은 전환 상품 보러가기 링크 : 클릭 시 전환상품 리스트로 이동 

- 고객 후기 : 고객 후기 중 메인 노출 여부가 체크된 게시물을 최신 등록 순으로 최대 6건까지 노출하며, '베스트'로 체크된 게시물 우선 노출. 
  ㄴ 작성 구분 탭 (해당 탭별 노출 설정한 고객 후기 없는 경우 해당 탭 비노출 처리)
      : 전체(디폴트) 
      : '상조 상품' 탭 : 상조 상품 가입 상담 후기, 상조 상품 이용 후기  
      : '전환 상품' 탭 : 전환 상품 가입 상품 후기, 전환 상품 이용 후기
      : '제휴 혜택' 탭 : 제휴 혜택 이용 후기
  ㄴ 고객 후기 리스트
      → 작성 구분 상세
          : 상조 상품 가입 상담 후기
          : 상조 상품 이용 후기
          : 전환 상품 가입 상담 후기
          : 전환 상품 이용 후기
          : 제휴 혜택 이용 후기
      → 후기 상품명 : 상조 상품명 or 전환 상품명 or 제휴 혜택명
          : 대상 후기 상품이 존재하는 경우 노출
      → 베스트 아이콘
      → 만족도 
      → 만족사항 : 최대 2개까지만 노출, 선택한 나머지 만족사항은 더보기 통한 후기 리스트팝업 통해 확인
      → 내용
      → 내용 더보기 : 해당 후기내용 확인 가능한 후기 리스트팝업 호출, 동일 탭 내 다른 후기들도 좌우슬라이드로 확인 
      → 등록일
      → 작성자 이름(마스킹 처리) 
  ㄴ 더 많은 후기 보러가기 링크 : 상단 선택한 작성 구분 탭 기준 전체 고객 후기 페이지로 이동


- 장례 정보 영역 : 관리자 공통코드에 등록된 장례정보 카테고리 중에서 ‘노출’상태인 공통코드를 우선순위가 높은 순으로 노출. 
  ㄴ 카테고리명 : 관리자 등록 공통코드명 호출 
  ㄴ 카테고리 서브설명 : 장례정보의 카테고리코드별 속성 1에 입력된 값을 호출
  ㄴ 배너 영역 클릭 시 해당 장례정보 카테고리 리스트로 이동 

- 퀵링크 영역 (하드코딩영역)
  ㄴ 부고알림 서비스 바로가기 
  ㄴ 고객지원 퀵링크 (자주하는 질문, 1:1문의, 공지사항) 
  ㄴ 고객센터 안내 및 24시 긴급 장례 접수/상담 전화 클릭 시 전화 연결

#### 기능 설명

1 전체 모달(레이어)팝업 
- 팝업 관리 중 팝업 구분이 '레이어 팝업', 조회시점의 노출 기간에 해당하고, 노출 여부가 ' 노출'이며, 노출 우선 순위가 가장 낮은 팝업 정보를 메인 페이지 랜딩 시 노출
  ㄴ 레이어 팝업 노출 개수 2개 이상인 경우 좌/우 슬라이드 처리 및 인디케이터 노출
- 본문 : 등록 이미지 및 링크 노출, 적용
- 닫기 : 레이어 팝업 닫기
- 오늘 그만보기 : 해당 접속자에게 1일간 팝업 출력 중지

2. 하단 토스트 팝업
- 팝업 관리 중 팝업 구분이 '하단 레이어 팝업', 조회 시점의 노출 기간에 해당하고, 노출 여부가 '노출'이며, 노출 우선 순위가 가장 낮은 팝업 정보를 메인 페이지 랜딩 시 노출
  ㄴ 하단 레이어 팝업 노출 개수 2개 이상인 경우 좌/우 슬라이드 처리 및 인디케이터 노출
- 본문 : 등록 이미지 및 링크 노출, 적용
- 닫기 : 하단 레이어 팝업 닫기
- 오늘 그만보기 : 해당 접속자에게 1일간 팝업 출력 중지

3. 레이어 팝업과 하단 레이어 팝업의 진행기간 중복 시 레이어 팝업 > 하단 레이어 팝업 출력
(레이어 팝업과 하단 토스트 팝업의 진행기간 중복 시 레이어 팝업> 하단 토스트 팝업 출력(레이어 닫히면, 그 다음 순서로 하단토스트팝업 노출)

#### 기능 설명

1. 로그인(계약 회원) 대상 별도 노출 영역
- 기가입자
  ㄴ 알림 배치 : 정책으로 정의된 알람을 해당 시기에 노출함 
      → 고객 후기 작성 알림 (클릭 시, 후기 관리 페이지로 이동) 
          : 상조 상품 가입 상담 후기 (해당 후기 등록 미완료 상태 및 후기 등록 가능한 경우 노출)
          : 상조 상품 이용 후기 (해당 후기 등록 미완료 상태 및 후기 등록 가능한 경우 노출)
          : 전환 상품 가입 상담 후기 (해당 후기 등록 미완료 상태 및 후기 등록 가능한 경우 노출)
          : 전환 상품 이용 후기 (해당 후기 등록 미완료 상태 및 후기 등록 가능한 경우 노출)
          : 제휴 혜택 이용 후기 (해당 후기 등록 미완료 상태 및 후기 등록 가능한 경우 노출)
      → 온라인 계약 미완료 상태 알림
          : 당일 기준 온라인 계약 상품 미완료 상태인 경우 (클릭 시 진행중 온라인 계약 단계 페이지로 이동)
      → 상담 신청 완료 상태 알림
          : 상담 상태값 '완료' 처리 시 비노출 
          : 상담 희망 날짜/시간 이후는 비노출 
      → 1:1 문의 등록/답변 알림
          : 1:1 문의 등록 완료 상태 알림 (클릭 시, 마이 > 1:1 문의 내역 페이지로 이동) 
          : 1:1 문의 답변 등록 완료 상태 알림 (클릭 시, 마이 > 1:1 문의 내역 페이지로 이동) 
      → 전자서명 대기 알림 
          : 전자계약 신청완료(0회차 결제완료하여 리스트 조회되는 시점) 시, 노출. (클릭 시 계약조회 리스트로 이동) 
  ㄴ 알림 노출 조건 정의 
      → 이용후기 알림 
          : 각 후기 알림노출조건별로 알림 노출, 이용후기 작성기한(3개월) 경과 시 삭제 또는 해당 후기 등록되면 삭제 
      → 예약상담신청알림 
          : 최대한 빨리 예약 - 예약신청완료 시 알림 노출, 상담완료값 받으면 삭제     
          : 날짜/시간선택 예약 - 예약신청완료 시 알림 노출, 예약상담일시 지나면 삭제 
      → 1:1문의 등록 알림 
          : 1:1 문의 등록 알림 - 문의글 등록 완료 시 알림 노출, 알림을 확인하거나 해당 문의 답변 완료상태값 받으면 삭제     
          : 1:1 문의 답변 알림 - 1:1 문의 답변상태로 처리 시 알림 노출, 답변을 확인하거나 1개월 후 삭제 
      → 온라인계약 미완료 상태 알림 : 당일기준 미완료 상태 계약 존재하는 경우 알림 노출, 알림을 확인하거나 당일이 지나면 삭제 
      → 전자서명 대기 알림 : 온라인 상품가입 신청 완료(0회차결제완료하여 리스트조회되는시점)시 알림 노출, 알림을 확인하거나 전자계약서명대기일이 지나면 삭제
  ㄴ 이용 중인 상품 노출
      → 계약 상태가 '정상'인 제품을 최신 계약순으로 노출 (클릭 시 계약조회 상세 이동)
          : 상품명
          : 다음 회차 납입 예정일 (마지막 납부 회차 및 거치 기간에 해당하는 경우 "없음" 으로 표기)
          : 다음 회차 납입 예정 금액 (마지막 납부 회차 및 거치 기간에 해당하는 경우 "없음" 으로 표기)

  ㄴ 작성 구분 탭 (해당 탭별 노출 설정한 고객 후기 없는 경우 해당 탭 비노출 처리)
      → 전체(디폴트) 
      → '상조 상품' 탭 : 상조 상품 가입 상담 후기, 상조 상품 이용 후기  
      → '전환 상품' 탭 : 전환 상품 가입 상품 후기, 전환 상품 이용 후기
      → '제휴 혜택' 탭 : 제휴 혜택 이용 후기
  ㄴ 고객 후기 리스트 (디폴트 20건 노출) 
      → 작성 구분 상세
          : 상조 상품 가입 상담 후기
          : 상조 상품 이용 후기
          : 전환 상품 가입 상담 후기
          : 전환 상품 이용 후기
          : 제휴 혜택 이용 후기
      → 후기 상품명 : 상조 상품명 or 전환 상품명 or 제휴 혜택명
          : 대상 후기 상품이 존재하는 경우 노출
      → 베스트 아이콘
      → 만족도
      → 만족사항
      → 내용
      → 등록일
      → 작성자 이름(마스킹처리) 
 ㄴ 도구 : 더보기 버튼 - 추가 10개 게시물 호출
